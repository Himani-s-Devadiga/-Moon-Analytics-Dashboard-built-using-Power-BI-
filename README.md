# Moon-Analytics-Dashboard-built-using-Power-BI
🚀 ** Moon Analytics Dashboard built using Power BI & DAX!** 🌙

In this project, I explored lunar data to analyze moon phases, eclipses, and astronomical patterns through interactive visualizations.

🔹 **What I worked on**
• Analyzed lunar events and moon phases using Power BI and DAX
• Built latitude-based maps to visualize lunar and eclipse visibility
• Calculated occurrences of Blue Moons, Red Moons, Supermoons, and regular moon phases
• Explored moonrise and moonset patterns, lunar distance, and eclipse trends
• Designed interactive dashboards to reveal patterns and insights in space data

💡 **Key Learnings**
• Advanced data modeling and visualization with Power BI
• Extracting meaningful insights from complex datasets
• Data storytelling through visual analytics
• Combining data science with my passion for astronomy

This project strengthened my skills in **data analytics, dashboard design, and insight generation** while exploring fascinating space data. 🌌
